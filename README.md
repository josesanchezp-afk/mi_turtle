# mi_turtle

Este proyecto contiene scripts simples en Python que simulan el movimiento de una tortuga en la consola. Los scripts permiten ingresar pasos y giros para representar el movimiento de la tortuga de manera textual.

- `Untitled-1.py`: Script b√°sico para movimiento horizontal de la tortuga.
- `Untitled-2.py`: Script para movimiento vertical de la tortuga.
- `Untitled-3.py`: Script con movimiento horizontal y vertical.
- `Untitled-4.1.py`: Variante adicional.
- `Untitled-5.py`: Otra variante.

## Uso

Ejecuta cualquiera de los scripts con Python y sigue las instrucciones para ingresar los pasos.
